Inspetor de elementos

(Voltar)

O inspetor de elementos do navegador é uma excelente ferramenta voltada para aqueles que possuem

interesse em aprender mais sobre desenvolvimento web, identificando como uma página é escrita e como

ela está organizada.

Estrutura básica do html

(Voltar)

Essa linguagem de marcação ela é definida da seguinte forma:

Inicialmente, todo o desenvolvimento com essa linguagem deve conter as principais instruções e tags, sendo elas:

"DOCTYPE, HTML, HEAD, META, TITLE E BODY"

A instrução doctype é a primeira que obrigatoriamente deve ser utilizada para

que seu navegador seja capaz de reconhecer a linguagem. Já a tag html é aquela responsável por marcar o início e o fim do código.

quando se trata da tag head podemos entender que ela desempenhará a função de cabeçalho do documento.

Nesse contexto, a tag meta irá especificar o conjunto de caracteres usados para renderizar o conteúdo do texto.

De modo simplificado, title é a tag que torna possível a existência de um título na página.

Por fim, e não menos importante, a tag body é aquela que marcará o espaço onde estará incluso o conteúdo digital.

Por que html não é uma linguagem de programação?

(Voltar)

O html não é considerado uma linguagem de programação, pois ela não tem essa capacidade de criar funcionalidades dinâmicas. Ela é caracterizada somente como uma linguagem voltada para a construção de páginas na internet.

O navegador se faz responsável pela interpretação desse conteúdo. A sigla HTML do inglês HyperText Markup Language,

significa no português, linguaguem de marcação de hipertexto.
